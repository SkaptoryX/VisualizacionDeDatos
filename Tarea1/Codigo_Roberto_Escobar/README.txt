Los datos utilizados en este proyecto provienen del dataset:
"LeMans 24h Race Winners", disponible en Kaggle.
(https://www.kaggle.com/datasets/erykwitkowski/lemans-24h-race-winners?select=data.csv)

Este conjunto de datos contiene información histórica sobre carreras, pilotos, escuderías, velocidad promedio, cantidad de vueltas, entre otras.
